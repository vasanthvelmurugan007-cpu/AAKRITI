3. HOSTING GUIDE & COST ESTIMATES
==================================================

Now that the code is ready, the next step is "Hosting"â€”making the site accessible to the world via a public URL (e.g., www.aakritii.org).

A. WHAT YOU NEED
----------------
1. **Domain Name:** The address of your website (e.g., .org, .com).
2. **Hosting Server:** A computer in the cloud that runs your website 24/7.
3. **Database/Storage:** A place to keep your text data and images (Cloudinary).

B. COST BREAKDOWN (Estimates)
-----------------------------
1. **Domain Name:** ~ $12 - $20 per year. (One-time annual purchase).
   - *Where to buy:* GoDaddy, Namecheap, or Google Domains.

2. **Hosting Options:**

   *OPTION 1: FREE / LOW COST (Recommended for starting)*
   - **Frontend (The Visuals):** Vercel or Netlify (Free Tier).
   - **Backend (The Logic):** Render or Railway (Free Tier available, or ~$5/month for better speed).
   - **Images:** Cloudinary (Free Tier is very generous and likely sufficient for years).
   - **TOTAL COST:** $0 - $5 per month.

   *OPTION 2: PREMIUM PERFORMANCE (Recommended for high traffic)*
   - **DigitalOcean or AWS:** Full control over a virtual server.
   - **TOTAL COST:** ~$10 - $20 per month.

C. HOW TO BUY & LAUNCH
----------------------
1. **Purchase Domain:** Go to Namecheap or GoDaddy and buy your desired name.
2. **Sign Up for Hosting:**
   - Go to [vercel.com](https://vercel.com) for the frontend.
   - Go to [render.com](https://render.com) for the backend.
3. **Connect Code:** You (or we) will upload the code to a GitHub repository, and then connect that repository to Vercel/Render. They will automatically pull the code and build the site.
4. **Configure DNS:** Point your new Domain Name to the Vercel/Render servers (changing "A Records").

We have prepared all the necessary configuration files (`vercel.json`, `render.yaml`, `Procfile`) inside the project folder so that this process is as smooth as possible.
